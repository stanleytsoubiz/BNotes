## B3 通用升級檢查表

每篇升級時，B3 應依序確認：

1. **frontmatter** 是否符合 v1.5（status / category / b1_reviewed_at / gm_approved_at）
2. **key-box**：3-5 條核心數字/事實，可掃讀
3. **geo-box**：FAQ 形式 3-5 問，每問 40-80 字直接答案，方便 ChatGPT/Perplexity 引用
4. **references block**：列出 3-7 個權威來源（SCA / CQI / WCE / 學術論文 / 業界白皮書）
5. **「我們」語氣**：通篇檢查第一人稱單數→團隊視角
6. **反常識切入點**：是否包含至少一個讀者讀完想轉發的反常識事實（社群貨幣）
7. **內部連結**：與既有文章建立至少 2 個 internal link（ZK 原則）
