Output Task 3 · Growth System

Newsletter 完整成長系統

🧲 訂閱引流策略：Lead Magnet 設計建議

核心原則：Lead Magnet 必須是「立刻可用的實用資源」，而非泛泛的電子書。針對咖啡愛好者，以下四個方向最有吸引力：

📊
手沖參數對照表 PDF
整理 V60 / Kono / Origami / 波浪濾杯的標準參數（水溫、比例、研磨度、時間），可貼在沖煮區旁使用。
→ 最高轉化：實用工具型
🗺️
咖啡風味輪 36 味解析
將抽象的咖啡風味描述對照實際風味來源（水果酸 / 堅果 / 焦糖），附帶「如何在杯中找到它」的初學者指南。
→ 知識型：高收藏率
⚗️
30 天咖啡實驗挑戰表
每天嘗試一個沖煮變數（水溫 ±5°C、研磨粗細、注水方式），記錄風味差異，培養科學化喝咖啡習慣。
→ 互動型：提升留存率
🏪
台灣精品咖啡館地圖 2026
精選全台 50 間值得一訪的精品咖啡館，附帶特色簡介與交通資訊，可搭配 Google Maps 清單使用。
→ 生活型：社群分享率最高

📍 Lead Magnet 觸點策略

  • IG 貼文留言「手沖表」→ ManyChat 自動私訊 PDF 下載連結 + Beehiiv 訂閱頁
  • YouTube 影片說明欄：「免費下載 [對照表] → bnotes.tw/free」
  • Blog 文章底部：嵌入 Beehiiv 訂閱表單，顯示 Lead Magnet 預覽圖
  • Podcast 每集開頭：「本集附贈 [XX資源]，加入 BNotes Newsletter 免費取得」

月度 Newsletter Skywork 生成 Prompt 模板

每個月底，將以下 Prompt 複製到 Skywork，填入當月變數，生成完整 Newsletter 草稿。預計生成時間：3 分鐘。

# BNotes 月度 Newsletter 生成 Prompt # 版本：2026 v1.0 | 每次使用前更新 [變數] 欄位 你是 BNotes 的內容策略師與主筆，BNotes 是一個台灣精品咖啡知識品牌， 風格定位：極簡 × 科學 × 有溫度，語氣：知識型但不冷漠，像一位懂咖啡的好友在寫信給你。 ## 本期變數 - 發送日期：[YYYY年MM月DD日，週三] - 本月主題柱：[例：手沖咖啡 / 咖啡豆產地 / 義式咖啡機] - 核心關鍵詞：[3 個長尾詞，例：V60 vs Origami / 淺焙養豆時間 / 厭氧發酵味道] - 本期 Lead Story：[本月最重要的一則咖啡知識 / 實驗發現 / 產業新聞] - 特別企劃（選填）：[如：2026 世界百大咖啡館 / 本月新品豆測評] - 讀者分眾：[全體訂閱者 / brew_pourover 標籤 / brew_espresso 標籤] - CTA 連結：[本期指向的 Blog 文章 URL / IG 貼文 URL] ## 請生成以下 Newsletter 結構 【信件主旨】（提供 3 個選項，含問句型/陳述型/數字型，供 A/B 測試使用） 【Preview Text】（40 字以內，補充主旨信息，提升開信率） 【開場白】（3-4 句，以一個生活場景或感官描述開頭，帶出本期主題，親切自然） 【本期重點：Lead Story】（300-400 字） - 核心知識點（用數據或實驗支撐） - 為什麼這對讀者重要（實際應用場景） - 一個可立刻嘗試的行動建議 【BNotes 精選：3 則快讀知識】（每則 60-80 字，格式：emoji + 標題 + 內容） 【本週 IG 精選】（2-3 句，介紹本週 IG 上最受歡迎的貼文，附說明與連結文字） 【讀者問答（Q&A）】（選填：本月精選一個來自讀者的問題並給出詳細解答） 【CTA 區塊】（清晰的單一行動呼籲 + 按鈕文字建議） 【結語】（2 句，溫暖收尾，預告下期主題，讓讀者期待） ## 格式要求 - 字數：全文 800-1200 字（繁體中文） - 段落：每段不超過 4 行，使用 emoji 輔助視覺呼吸 - 語氣：避免行銷話術，像在寫給朋友的信 - SEO：自然融入本期核心關鍵詞（不硬塞） - 避免：「絕對」「保證」「最好」等誇大詞彙

  • 排版加入 Beehiiv AI Image Tool 生成的配圖

  • 發送後 48 小時，記錄開信率/CTR，更新分眾標籤
