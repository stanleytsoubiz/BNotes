01
圖片情緒分類系統
每篇文章 Hero 圖必須歸入以下四種情緒類型之一
🔬
精準型（Precision）
強調技術、科學與工藝的精準度。高對比、冷色調偏移、器材特寫或萃取細節。
手沖技法 義式萃取 器材測評 科學解析
🌿
沉浸型（Immersive）
廣角環境、自然風光、產地人物。讓讀者感受「置身其中」的空間感與故事性。
產地報導 農場故事 風土科學 城市地圖
☀️
溫暖型（Warmth）
柔焦、暖色、生活場景。喚起閱讀咖啡的舒適與日常愉悅感。
節氣/季節 生活儀式 禮品指南 閱讀配對
🏆
探索型（Discovery）
知識感強、層次豐富、對比明顯。適合年報、白皮書、競賽解析等重磅文章。
年度報告 競賽解析 白皮書 歷史深探

圖片比例規範
嚴格執行統一比例，確保全站版面整潔一致
Hero Banner 16 : 9
Hero 圖片區域（16:9）
  • 最小尺寸：1200 × 675px
  • 建議尺寸：1920 × 1080px
  • AI 生成時指定 aspect_ratio: 16:9
  • CSS object-fit: cover，焦點置中
  • 適用：所有文章的 .article-hero
文中副圖 4 : 3
文中插圖區域（4:3）
  • 最小尺寸：800 × 600px
  • 建議尺寸：1200 × 900px
  • AI 生成時指定 aspect_ratio: 4:3
  • CSS object-fit: cover，圓角 .5rem
  • 適用：正文段落內所有 img 標籤

圖片選用規則表
明確列出允許、謹慎、禁止的圖片選用行為
情境判斷說明
圖片主題與文章內容直接相關✓ 允許最佳狀態，無需修改
使用 AI 生成定制圖✓ 優先旗艦文章最佳選擇，確保獨特性
Unsplash 主題精選圖（非重複）✓ 允許一般文章的標準做法
排版型 Hero（年報/白皮書類）✓ 優先無圖時的最佳設計選項
圖片在全站出現超過 2 次⚠ 謹慎必須替換其中一篇的圖片
圖片色調與品牌調色板衝突（高飽和藍/紫）⚠ 謹慎需加品牌色疊加 CSS 調和
使用 og-default.jpg 佔位圖✗ 禁止必須替換為主題圖或排版 Hero
Unsplash 咖啡豆特寫（已用超過 5 篇）✗ 禁止視覺重複性已達飽和
圖片尺寸低於 800px 寬✗ 禁止Retina 螢幕下會模糊
Hero 圖不符合 16:9 比例✗ 禁止CSS 會強制裁切，可能影響構圖
圖片包含浮水印或文字覆蓋✗ 禁止影響品牌專業度

07
圖片發布前核查清單
每篇新文章上線前，逐一確認以下項目
  • Hero 圖已替換，不使用 og-default.jpg
  • 圖片尺寸 ≥ 1200px 寬，符合 16:9 比例
  • 圖片主題與文章內容直接相關
  • 圖片已歸入四種情緒類型之一
  • 主色調與 BNotes 品牌調色板和諧
  • 全站無重複使用同一張圖（超過 2 篇）
  • og:image meta tag 已更新為新圖 URL
  • 品牌色疊加 CSS 已套用（mix-blend-mode）
  • 文中副圖確認比例接近 4:3
  • 圖片無浮水印、無外文文字覆蓋
